# Moodify API

Проект для классификации настроения аудио-треков с помощью FastAPI. API принимает аудио-файлы (MP3), извлекает признаки и предсказывает настроение трека с помощью ML-модели. Поддерживает историю запросов в SQLite БД и статистику запросов

## Запуск
1. Установите зависимости: `pip install -r requirements.txt`
2. Запустите сервер: `uvicorn app.main:app --reload` или `python -m app.main`
3. Откройте http://127.0.0.1:8000/docs для Swagger UI

## Структура проекта
- **`requirements.txt`**: Основные библиотеки с версиями
- **`app/`** (папка с основным кодом):
  - `main.py`: Основной файл FastAPI — определяет приложение, эндпоинты
  - `extract_audio_features.py`: Функции для извлечения аудио-признаков (MFCC, chroma, tempo и т.д.) из файла с помощью librosa и tsfel
  - `request_stats.py`: Логика для сбора и расчёта статистики запросов (время обработки, размеры файлов)
  - `models.py`: Pydantic модели для валидации данных
  - `database.py`: Настройка БД и функция `history_to_db` для сохранения истории
  - `ml_model.py`: Загрузка ML-модели (`moodify_knn_model.pkl`) и функция `get_prediction` для предсказания настроения
