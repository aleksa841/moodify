# 🎵 ИИ2025: Годовой проект

## 📌 Тема проекта
Сервис музыкальных рекомендаций под настроение и контекст

## 📝 Описание проекта
Проект направлен на разработку ml-модели, которая сможет классифицировать песни по их тематике и настроению на основе текстов, аудиофич и других характеристик

Применение модели:
- музыкальные рекомендации
- автоматическая генерация плейлистов

## 🎯 Цель проекта
Создать рекомендационную систему, которая помогает пользователям находить музыку в соответствии с их настроением и предпочтениями
